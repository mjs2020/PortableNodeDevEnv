Console Portable-Specific Issues

  • Portable Scripts — Console Portable modifies the Windows %PATH% environment variable to include the default Console Portable scripts directory.
    	\ConsolePortable\Data\scripts\
    To view a list of the currently available scripts type the following command into the console window: 
    	MyScripts
    Console Portable comes with the following default commands:
    • DelEmptyFolders - Deletes empty folders from either the current directory or the specifed directory
    • DelThumbs - Removes "thumbs.db" files from current directory
    • ThisComputer - Displays information about the current computer
    • Whois - NirSofer CLI app for whois lookup
    • WhosIP - NIRSofer CLI app for domain information lookup

  • Running a script when console starts or with creation of a new tab. — To do this you need to go in to the settings menu. This can be access via the menu at Edit > Settings...
    • On Startup
      Select "Console" from the left pane and type the following in the "Shell" text: 
      %COMSPEC% /k "x:\Path\To\Script.bat"
    • New Tab
      Select "Tabs" from the left pane and then select the "Tab Name" you wish to use from the list. Type the following in the "Shell" text: 
      %COMSPEC% /k "x:\Path\To\Script.bat"
    %COMSPEC% can be replaced with any command line shell you whish to use. If you use the drive letter that Console Portable is currently running on it will be updated the next time Console Portable is run.
